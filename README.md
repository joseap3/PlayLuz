# 🎵 PlayLuz

**PlayLuz** é um player de música PWA (Progressive Web App) moderno e instalável.

## ✨ Funcionalidades
- Player completo com visualizador de frequências
- Biblioteca de músicas com IndexedDB (banco local offline)
- Playlists personalizadas
- Seleção múltipla de músicas
- Equalizador (Grave, Médio, Agudo)
- Favoritos e Histórico
- Módulo de aprendizado musical (Quiz, Flashcards, Teoria)
- Modo Claro e Escuro
- Funciona 100% offline
- Instalável no Android, iOS, Windows e Linux

## 🚀 Como usar
Acesse via GitHub Pages e instale como PWA no seu dispositivo.

## 📲 Instalação
- **Android:** Chrome → menu ⋮ → "Adicionar à tela inicial"
- **iPhone:** Safari → Compartilhar ⬆ → "Adicionar à Tela de Início"
- **Windows/Linux:** Chrome/Edge → ícone ⊕ na barra de endereço
